# 指标管理模块接口文档

## 一、指标管理

### 1. 分页查询指标列表

**接口地址**：`GET /api/v1/indicators/list`

**请求参数**：

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| keyword | String | 否 | 搜索关键词（指标名称或编码） |
| pageNum | Integer | 否 | 页码，默认 1 |
| pageSize | Integer | 否 | 每页条数，默认 10 |

**响应示例**：
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "list": [
      {
        "id": 1,
        "indicatorName": "近12个月下游销售金额",
        "indicatorCode": "INV_F1_12M_DOWN_SALE_JSHJ_SUM",
        "indicatorLogic": "统计近12个月下游客户销售发票的含税金额总和",
        "taskId": 7,
        "createTime": "2026-05-15T10:00:00",
        "updateTime": "2026-05-15T10:00:00"
      }
    ],
    "total": 1,
    "pageNum": 1,
    "pageSize": 10
  },
  "timestamp": 1778816000000,
  "success": true
}
```

## 三、业务数据格式说明

### 指标数据结构（方案B）

每个指标的数据都是嵌套对象结构：

```json
{
  "inv_f1_12m_down_sale_jshj_sum": {
    "value": 123456.79,
    "source": "taskId=7",
    "raw_data": {
      "indicatorCode": "inv_f1_12m_down_sale_jshj_sum_teach",
      "sale_jshj_sum": 123456.79,
      "appDate": "2024-02-06",
      "windowRule": "近1-12个月(不含当月)",
      "collectionId": 2021827454116700214,
      "parsingId": 32,
      "processingId": 25
    },
    "collectionId": 2021827454116700214,
    "enterpriseName": "cq",
    "taxNo": "91500000MA5U123456",
    "taskId": 7
  },
  "tax_rate_6m": {
    "value": 3.5,
    "source": "taskId=3",
    "raw_data": {},
    "collectionId": 2021827454116700222,
    "enterpriseName": "测试企业名24",
    "taxNo": "91500000MA5U123456",
    "taskId": 3
  }
}
```

### 取值规则

- 在判断条件和公式中，使用 `指标编码.value` 获取实际数值
- 例如：`inv_f1_12m_down_sale_jshj_sum.value`
