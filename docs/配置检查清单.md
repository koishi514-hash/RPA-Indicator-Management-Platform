## 八、常见问题排查

### Q: Coze API调用失败
A: 检查以下几点：
1. 环境变量 `COZE_API_KEY` 是否正确配置
2. API Key 是否有访问权限
3. 网络连接是否正常

### Q: 数据库字段名不匹配
A: 确认以下几点：
1. 已执行最新的 SQL 脚本
2. 字段名已更新为 `coze_app_id` 和 `coze_api_key`
3. 实体类字段已同步更新

### Q: 额度计算结果解析错误
A: 检查以下几点：
1. Coze Bot 提示词配置正确
2. 输出模板格式正确（JSON）
3. 业务数据包含所有必需变量
